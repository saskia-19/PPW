CSS (Cascading Style Sheets) merupakan bahasa untuk mendesain dan mengatur tampilan halaman web yang memungkinkan pengembang web untuk mengubah warna, tata letak, font, dan elemen visual lainnya pada halaman HTML tanpa mempengaruhi struktur HTML itu sendiri [1]. CSS digunakan dalam HTML untuk mengatur layout halaman web, seperti penempatan elemen dengan flexbox atau grid, serta mengubah tampilan visual seperti warna, font, dan ukuran. CSS juga memungkinkan pembuatan halaman responsif yang menyesuaikan tampilan dengan ukuran layar perangkat menggunakan media query. Selain itu, CSS meningkatkan pengalaman pengguna dengan animasi, transisi, dan efek hover yang membuat halaman lebih interaktif dan menarik.
